# LazyDaily - Diary Mobile Application 
LazyDaily is a minimalist diary application designed to simplify the daily journaling experience. Born from the vision of creating a Comfort Zone for individuals seeking a private space to release suppressed emotions and navigate difficult days, the app offers a comforting platform for honest and personal self-expression.
Beyond its clean and simple interface, LazyDaily features an intuitive emoji-based mood tracking system and customizable privacy settings for each entry. To help users gain deeper insights into their emotional well-being, the app calculates weekly mood statistics, transforming daily feelings into clear and meaningful percentages.
